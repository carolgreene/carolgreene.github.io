I've been making Cheatsheets for different aspects of JavaScript  to help me remember  them better. Listed below are the methods from my String Methods Cheatsheet. All string methods return a new value. They do not alter the original variable.



...........Method................

* charAt()................Returns the character at the specified index
* charCodeAt().....Returns the Unicode of the character at the specified index
* concat()...............Joins two or more strings, and returns  a new joined string
* endsWith()...........Checks if a string ends with a specifed string/characters
* fromCharCode()..Converts Unicode values to characters
* includes().............Checks if a string contains the specified string/characters
* indexOf()..............Returns the index of the 1st occurance of specific value 
* lastIndexOf()......Returns the index of the last occurance of specific value
* localeCompare()..Compares two strings in the current locale
* match()....Searches a string and returns a match against regular expression 
* repeat().......Returns new string with a set number of copies of existing string
* replace().....Searches for specific value, replaces it and returns new string 
* search()......Searches for specific value and returns the position of the match
* slice()..........Extracts part of a string and returns a new string
* split()..........Splits a string into an array of substrings
* startsWith()...Checks to see if a string begins with specified characters
* substr().....Extracts set number of characters from string at set start position
* substring()....Extracts characters from string between 2 specified indices
* toLocaleLowerCase()...Converts string to lowercase, based on host's locale
* toLocaleUpperCase()...Converts string to uppercase, based on host's locale
* toLowerCase()......Converts a string to lowercase
* toUpperCase()......Converts a string to uppercase
* trim()................Removes whitespace from both ends of a string
* valueOf()........Returns the primitive value of a String object

These methods allow you to manipulate strings in a lot of different ways!
